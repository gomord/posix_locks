        Eli Bendersky Mod Dan Liew • a year ago

        No, nothing runs before relocations. But nothing is run to initialize myglob either. This is not "initializers" in the sense of running C++ constructors. POD types get their initialization by the compiler (by filling in the data section), without running any code at load time.
